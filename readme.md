# Avenir Digital

A small listing app for users based on `https://randomuser.me/documentation` for (ANDROID, IOS).

## Packages
- react-navigation v6
- axios v0.21.1
- date-fns v2
- redux v4
- redux-sage v1
- @reduxjs/toolkit v1
